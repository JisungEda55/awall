Installation Steps

First, define your wallpaper settings. Then create an automation to run daily. Finally, add the shortcut actions to update your lock screen.

Your Birthday

2

Create Automation

Open Shortcuts app → Go to Automation tab → New Automation → Time of Day → 6:00 AM → Repeat "Daily" → Select "Run Immediately" → "Create New Shortcut"

3

Create Shortcut

ADD THESE ACTIONS:

  1. "Get Contents of URL" → paste the following URL (완료 후 생성된 URL 입력)
  2. "Set Wallpaper Photo" → choose "Lock Screen"
Important: In "Set Wallpaper Photo", tap the arrow (→) to show options → disable both "Crop to Subject" and "Show Preview"
This prevents iOS from cropping and asking for confirmation each time.

Installation Steps

First, define your wallpaper settings. Then set up MacroDroid to automatically update your wallpaper daily.

1

Define your Wallpaper

Your Birthday
Phone Model
2

Prerequisites

Install MacroDroid from Google Play Store.

3

Setup Macro

Open MacroDroid → Add Macro

Trigger: Date/Time → Day/Time → Set time to 00:01:00 → Activate all weekdays

4

Configure Actions

4.1 Download Image

  • Web Interactions → HTTP Request → GET
  • Paste the URL (Step 1 완료 후 생성)
  • Block next actions until complete
  • Save HTTP response to file: /Download/life.png

4.2 Set Wallpaper

  • Device Settings → Set Wallpaper
  • Choose Image and Screen
  • Enter: /Download/life.png
Important: Use the exact same folder and filename in both actions.
5

Finalize

Give the macro a name → Tap Create Macro
